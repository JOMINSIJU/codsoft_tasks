# 🤖 CodSoft Artificial Intelligence Internship

This repository contains the projects completed as part of the **CodSoft Artificial Intelligence Internship**. These projects demonstrate concepts in Artificial Intelligence, Machine Learning, Computer Vision, and Game Theory using Python.

---

## 📂 Projects

### 🎮 Task 1 – Tic-Tac-Toe AI

An unbeatable Tic-Tac-Toe game powered by the **Minimax Algorithm**.

**Features**
- Human vs AI gameplay
- Unbeatable AI
- Input validation
- Play again option

**Technologies**
- Python
- Minimax Algorithm
- Game Theory

---

### 🎬 Task 2 – Movie Recommendation System

A Content-Based Movie Recommendation System that suggests similar movies using machine learning techniques.

**Features**
- Movie search
- Top 5 recommendations
- Content-Based Filtering
- Cosine Similarity

**Technologies**
- Python
- Pandas
- NumPy
- Scikit-learn

---

### 👤 Task 3 – Face Detection

A real-time Face Detection application using OpenCV.

**Features**
- Live webcam detection
- Multiple face detection
- Real-time bounding boxes

**Technologies**
- Python
- OpenCV
- Haar Cascade Classifier

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/JOMINSIJU/codsoft_tasks.git
```

Navigate to the repository:

```bash
cd codsoft_tasks
```

Install the required libraries:

```bash
pip install -r Task2_Recommendation_System/requirements.txt
pip install -r Task3_Face_Detection/requirements.txt
```
