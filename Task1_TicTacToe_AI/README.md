# 🎮 Tic-Tac-Toe AI

## 📌 Overview
This project is a Tic-Tac-Toe game where a human player competes against an AI. The AI uses the **Minimax Algorithm**, making it impossible to beat when it plays optimally.

## 🚀 Features
- Human vs AI gameplay
- Unbeatable AI using Minimax
- Input validation
- Replay option
- Clean command-line interface

## 🛠️ Technologies Used
- Python 3
- Minimax Algorithm
- Game Theory
