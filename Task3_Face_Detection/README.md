# 👤 Face Detection using OpenCV

## 📌 Overview

This project is a real-time Face Detection application developed as part of the CodSoft Artificial Intelligence Internship.

The application uses OpenCV's Haar Cascade Classifier to detect human faces from a live webcam feed and display bounding boxes around detected faces in real time.

---

## 🚀 Features

- Real-time webcam face detection
- Detects multiple faces
- Draws bounding boxes around faces
- Fast and lightweight

---

## 🛠 Technologies Used

- Python
- OpenCV
- NumPy
- Haar Cascade Classifier
